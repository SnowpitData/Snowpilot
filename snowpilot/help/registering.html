Users may register for a new SnowPilot account and need to provide only your name, username, email and password.

After registering, you will be directed to enter your Snowpit Preferences. When you create a new snowpit, this will control such things as

  • Default State/Province/Country, and possibly even Range or Advisory Region
    This will place the google map, and populate the location fields, for new snowpits in the correct location.
    Default location will also affect how you enter DATE AND TIME.
    If you set this to a location outside the US, you should enter date in this format: dd/mm/yyyy , and time is represented in 24 hour format, with no am/pm designation ( e.g. 14:45 versus 2:45pm ).
    If your default location is inside the US, you should enter data in this format: mm/dd/yyyy , with month before day; and time in 12 hr format, with am/pm.
  • Default Language: Your default language will determine which language most pages of the website are displayed in. It will also determine the language used when entering a new snowpit profile; and the labels on snowpit profiles that you make.
  • Measure from Top, Measure from Bottom
    This is an important notation, use your preference. It is a default setting that can be adjusted when a new snowpit is created for each individual pit
  • Snowpit Depth Units: cm or inches
  • Elevation Units: Feet or Meters
    Elevation Units affects which background map you are provided
    If you choose feet, the background map will be provided from US Geological Survey mapping data, with contour lines specified in feet.
    If you choose meters, the background map will be provided from OpenTopos/Open Streets data, with contour lines indicated in meters.
    This affects the map used on the 'Create New Snowpit' page, where you can choose the latitude/longitude or UTM coords of a snowpit location.
  • Default hemisphere (N / S and E / W)
  • Temperature Units
  • Coordinate type (Lat/Long, UTM or MGRS)
  • Affiliation Type / Organization
    You can set this beginning with organization type, then the specific organization you'd like to join. Please join an existing organization, do not use ( a group ) unless you are certain it does not already exist in the system.
    Multiple group affiliations are possible
  • Fracture Category
  • Hardness Scale
  • Language settings
    Your default language for the SnowPilot website. If you find errors or un-translated text, please let us know at admin@snowpilot.org , with the correct translated words

The last one, Organization Affiliation, can be set to whichever Ski Patrol, Forecast Center, Educational or Reasearch group, or Guide Service. This will allow you to view some 'semi-private' snowpit profiles from other users in that group; and, you will be able to share your snowpit profiles with only members of your group, but not everyone, if you so choose. More about Organization affiliation and users.

All 'Measurement Unit Prefs' will need to be filled out in order to create a snowpit.
